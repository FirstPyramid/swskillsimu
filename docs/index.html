소울워커 스킬 시뮬레이터

이 시뮬레이터는 자바스크립트를 필요로 합니다.
또한 대부분의 브라우저에서 동작합니다. 원작자는 다음과 같은 환경에서 동작을 확인했습니다: Firefox v62, Chrome v69, Edge.

현재, 이 시뮬레이터는 스킬 미리보기 기능을 지원합니다(and use Promise object). 따라서 오래된 브라우저에서는 정상 동작하지 않을 수도 있습니다(Internet Explorer 11 등).

변경 내역 전체를 보려면 이곳으로 가십시오: Github Repository commit log.
